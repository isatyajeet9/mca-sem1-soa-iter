Q2. Why do we use JavaScript? Why is it so popular? (Advantages) Uses of JavaScript: 1 To make web pages interactive (buttons, forms, animations). 2 To validate form input before sending to the server. 3 To update content without refreshing the page (AJAX). 4 To build frontend frameworks (React, Angular, Vue). 5 To build backend servers using Node.js. 6 To create mobile apps, desktop apps, and even games. Advantages: 1. Fast execution - Runs directly in the browser without needing compilation. 2. Easy to learn - Simple and beginner-friendly syntax. 3. Platform independent - Works on all browsers and devices. 4. Huge community support - Many libraries and frameworks. 5. Full-stack development - * Frontend: React, Angular, Vue * Backend: Node.js 6. Rich interfaces - Allows animations, sliders, popups, dynamic content. 7. Asynchronous nature - Makes modern apps smooth using Promises, async/await. 8. Versatile - Can build websites, apps, servers, games, chat apps, etc.
